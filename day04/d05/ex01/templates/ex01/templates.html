The Template Engine

Django provides an easy way to create HTML templates while following the DRY (Donâ€™t Repeat Yourself) principle. Templates help separate the presentation layer from the application logic.

Blocks

The {% block %} tag defines a replaceable section within a base template. It allows child templates to override specific blocks while keeping the rest of the layout unchanged. This makes code more reusable and easier to maintain.

Loops

The {% for item in iterable %} tag is used to loop through data provided in the context. All content between {% for %} and {% endfor %} will be repeated once per element in the iterable.

If statements

The {% if %} tag allows conditional rendering inside a template. For example, {% if myvar == 1 %} displays the enclosed content only if the condition is true.

Context variables

The {{ myvar }} syntax displays the value of a variable passed from the view to the template. This is how dynamic data (like user names or messages) appears inside static HTML.
